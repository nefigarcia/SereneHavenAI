# **App Name**: SereneHaven Suite

## Core Features:

- Resident Profile Management: Centralized system to manage resident personal details, emergency contacts, room assignment, and basic care preferences using mock data for now.
- Staff Roster & Scheduling: Manage staff profiles, roles, contact information, and basic work schedules, integrating mock data.
- Billing & Payments Dashboard: Basic financial tracking for resident invoices, payments, and overdue amounts, currently supported by mock data.
- Health Log Entries: Staff can log daily health notes, medication administrations, and incident reports for each resident.
- Activity Planner & Announcements: Create and publish daily or weekly activity schedules and facility-wide announcements for residents and staff, displaying mock data initially.
- AI-Powered Personalized Updates Tool: A generative AI tool to assist staff in drafting compassionate and personalized update messages for family members based on logged health and activity events.
- Internationalization (i18n) Support: Implement core architecture for dynamic language switching, starting with English and full Spanish translation support if accessed from Mexico.

## Style Guidelines:

- Primary color: A sophisticated, calming deep blue (#2E60AD) to evoke trust and care.
- Background color: A very light, subtle grey-blue (#E7EDF0) to ensure readability and a serene feel.
- Accent color: A vibrant, clear cyan (#45CDE8) to highlight key actions and information, creating gentle contrast.
- Headlines font: 'PT Sans', a humanist sans-serif for its warmth and modern appeal. Body font: 'Inter', a neutral sans-serif for excellent readability in data-heavy sections.
- Utilize a consistent set of clean, modern line-based icons to represent various administrative, medical, and social functions, enhancing clarity and user experience.
- Employ a responsive, card-based layout with generous whitespace to ensure content is digestible and functions well across all mobile and desktop devices. Information hierarchy will be clear and intuitive.
- Subtle, professional transitions for page loads, state changes, and form submissions. Utilize minimal hover effects to guide user interaction without distraction.
